Q3a — One-Sample Tests
SPSS steps
Q3a(I) — All employees: salary vs 75k and 50k
Analyze → Compare Means → One-Sample T Test
1. Move Salary into Test Variable(s). Set Test Value = 75. Click OK.
2. Repeat with Test Value = 50.
Conclusion (75k): If p < 0.05, reject H₀ — average salary differs significantly from 75 thousand dollars. If p ≥ 0.05, insufficient evidence to reject; 75k is a plausible mean.
Conclusion (50k): Same logic. Since the sample mean is likely well above 50k, the test is expected to be highly significant (p < 0.05).
SPSS steps
Q3a(II) — 10% sample: salary vs 75k and 50k
1. Select 10% sample: Data → Select Cases → Random Sample → Approximately 10%. Click Continue → OK.
Analyze → Compare Means → One-Sample T Test
2. Move Salary into Test Variable(s). Test with 75 then 50. Click OK each time.
Conclusion: Same interpretation as Q3a(I). With a smaller sample, the test has less statistical power — you are less likely to detect a true difference. If p < 0.05, still reject H₀.
SPSS steps
Q3a(III) — 10% sample: house value vs 75k and 50k
1. Keep the 10% random sample selected (or re-select as above).
Analyze → Compare Means → One-Sample T Test
2. Move Value of House into Test Variable(s). Test Value = 75, then 50. Click OK.
Conclusion: If p < 0.05, average house value in the sample differs significantly from the test value. Interpret direction — if the sample mean exceeds the test value, house values tend to be higher than hypothesized.
SPSS steps
Q3a(IV) — Sick days, ID 1 to 25, test value = 10
1. Data → Select Cases → If condition is satisfied. Enter: ID <= 25. Click Continue → OK.
Analyze → Compare Means → One-Sample T Test
2. Move Sick Days into Test Variable(s). Set Test Value = 10. Click OK.
Conclusion: At α = 0.05, if p < 0.05 reject H₀ — average sick days for employees with ID 1–25 differ significantly from 10 days. If p ≥ 0.05, there is insufficient evidence to conclude the average differs from 10 days.

Q4 — Multiple Linear Regression
SPSS steps
Q4a — House value on salary & bedrooms
Analyze → Regression → Linear
1. Move Value of House into Dependent. Move Salary and Number of Bedrooms into Independent(s).
2. Click Statistics → check: Estimates, Confidence intervals, Model fit, R squared change, Descriptives, Part and partial correlations, Collinearity diagnostics. Continue → OK.
Prediction (salary=90, bedrooms=5)
3. From the Coefficients table, read the Constant (B₀), B₁ for Salary, and B₂ for Bedrooms. Compute: Value = B₀ + B₁(90) + B₂(5). Result ≈ 351.7 thousand dollars.
Correlation matrix
Analyze → Correlate → Bivariate
4. Move Value of House, Salary, Number of Bedrooms, Distance into Variables. Check Pearson. Click OK.
Conclusion (prediction): An employee earning 90k with 5 bedrooms is expected to have a house worth approximately 351.7 thousand dollars.
Conclusion (significance): Overall F p < 0.05 → model is statistically significant. Check individual predictor p-values — if p < 0.05, that predictor significantly contributes to house value.
Conclusion (R² & correlations): R² ≈ 0.509 means salary and bedrooms together explain 50.9% of variation in house value. Salary (r = 0.557) and bedrooms (r = 0.685) both show significant positive relationships with house value. Distance (r = −0.127, p = 0.061) is not significant.
SPSS steps

Analyze → Regression → Linear

2. Same Statistics options as Q4a. Continue → OK.
Partial & zero-order correlations
